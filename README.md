1. **Open the Application**
   - Open `index.html` in your web browser

2. **Add a Contact**
   - Enter name, email, and phone number in the form
   - Click "➕ Add Contact"
   - Contact appears in the list below

3. **Search Contacts**
   - Type in the search box to filter by name or email
   - Results update in real-time

4. **Edit a Contact**
   - Click ✏️ Edit button next to a contact
   - Update the information in the prompts
   - Changes are automatically saved

5. **Delete a Contact**
   - Click 🗑️ Delete button next to a contact
   - Confirm the deletion in the dialog

6. **Reorder Contacts**
   - Click and drag any contact to reorder the list
   - New order is automatically saved
